This is a small brakout board (10mm x 10mm) for use with the TI HDC1080 humidity and temperature sensor

There are pads on both sides of the board for a 100n ceramic capacitor, but only one needs to be populated depending on how the board is mounted.

Electrical connection is by four PTH on 0.05" centers. This makes it convenient to solder ordinary ribbon cable on either side of the board.

There are four mounting holes, 1mm in from each corner which allows heat staking to a 3D printed enclosure.
